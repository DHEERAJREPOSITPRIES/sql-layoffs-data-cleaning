# 📊 SQL Data Cleaning Project – Layoffs Dataset

## 🔍 Overview
This project demonstrates an end-to-end data cleaning pipeline using SQL on a real-world layoffs dataset.

## ⚙️ Steps Performed
- Removed duplicate records using ROW_NUMBER()
- Standardized text fields using TRIM
- Converted date column from TEXT to DATE
- Handled NULL and missing values
- Cleaned inconsistent categorical values (e.g., Crypto variations)
- Removed irrelevant rows

## 🛠️ Tech Stack
- MySQL
- SQL Window Functions

## 📁 Project Structure
- data/ → raw and cleaned datasets
- sql/ → data cleaning SQL script

## 🚀 Outcome
Cleaned dataset ready for analysis and visualization
